Overview

The Control Panel is the central hub for all administrative operations within the system. It provides direct access to the most important configuration and management areas, allowing administrators to understand the system structure and navigate efficiently without relying on specific routes or internal knowledge.

This section focuses on explaining how the Control Panel is organized, what each entry point represents, and how access is controlled.

Available Sections

From the Control Panel, administrators can access the following core areas:

  • User Management: Manage system users, review accounts, and control access at a high level.
  • Role Management: Define roles and determine which permissions are available to each role.
  • Company Management: Configure which modules and features are enabled per company.
  • Advanced Options: Access sensitive system-level tools intended for experienced administrators.

Each section represents a distinct responsibility area and is documented in detail in its own dedicated page.

Navigation Behavior

The Control Panel uses a card-based layout to present available sections. Each card acts as a direct navigation entry point and is displayed only if the current user has permission to access the corresponding module.

This approach ensures

  • A clean and uncluttered interface
  • Reduced cognitive load for new users
  • Visibility only of relevant administrative areas

Access Control and Visibility

Access to the Control Panel and its sections is governed by multiple layers:

  • Role permissions determine which sections are available
  • Company configuration may further restrict visibility
  • Module availability controls whether a section is shown at all

As a result, two users with the same role may see different options depending on the active company configuration.

Advanced Options Entry

The Control Panel includes an entry point to Advanced Options, which provides access to sensitive system-level features.

Before accessing this area, users are presented with a warning screen to confirm intent and reduce the risk of accidental changes. This behavior is intentional and designed to protect system integrity.

Detailed explanations of each advanced tool are covered in the Advanced Options documentation.
